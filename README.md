# build-powershell

[Atom Build](https://atombuild.github.io/) provider for Windows PowerShell, runs PowerShell scripts

## Installation

### apm

* Install package `apm install build-powershell` (or use the GUI)

### GitHub

1. Change directory `cd ~/.atom/packages/`
2. Clone repository `git clone https://github.com/idleberg/atom-build-powershell build-powershell`

## Usage

### Build

Before you can build, select an active target with your preferred build option.

Available targets:

* `PowerShell: Run script`

### Shortcuts

Here's a reminder of the default shortcuts you can use with this package:

**Choose target**

<kbd>Super</kbd>+<kbd>Alt</kbd>+<kbd>T</kbd> or <kbd>F7</kbd>

**Toggle build panel**

<kbd>Super</kbd>+<kbd>Alt</kbd>+<kbd>V</kbd> or <kbd>F8</kbd>

**Build script**

<kbd>Super</kbd>+<kbd>Alt</kbd>+<kbd>B</kbd> or <kbd>F9</kbd>
